# Student Management System

A simple Java console-based Student Management System using JDBC and MySQL.

## Features

- Add Student
- View Students
- Search Student by ID
- Update Student Details
- Delete Student

## Technologies Used

- Java
- JDBC
- MySQL

## Setup

1. Create a MySQL database.
2. Create the `students` table.
3. Set environment variables:
   - DB_URL
   - DB_USER
   - DB_PASSWORD
4. Run the project.
